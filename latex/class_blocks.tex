\doxysubsection{Detailed Description}
Manages Tetris pieces, including their shapes, rendering, and ghost pieces. 

The \doxylink{class_blocks}{Blocks} class defines the shapes of Tetris pieces as 4x4 matrices, provides methods to render the pieces on the game board, and supports ghost pieces to help players visualize where a piece will land.
