\doxysubsection{Detailed Description}
Manages the main game window and various game menus (Lobby, Match, Pause, Endgame). 

The \doxylink{class_window_manager}{Window\+Manager} class handles the creation and rendering of the game window, as well as the display and functionality of different game menus.
