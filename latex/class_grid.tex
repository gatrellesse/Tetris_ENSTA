\doxysubsection{Detailed Description}
Represents the main game grid for Tetris. 

The \doxylink{class_grid}{Grid} class handles the rendering of the game grid, the state of each cell, and operations such as line cleaning and grid resetting. It extends the \doxylink{class_window_manager}{Window\+Manager} class for managing rendering within a window.
