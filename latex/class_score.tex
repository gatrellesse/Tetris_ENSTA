\doxysubsection{Detailed Description}
Manages the scoring, level progression, and speed adjustments for the Tetris game. 

The \doxylink{class_score}{Score} class tracks the player\textquotesingle{}s current level, total points, and adjusts the game speed based on the player\textquotesingle{}s progression.
